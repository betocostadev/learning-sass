SASS Arrays - Creating List Elements

Using SASS when we create something like a variable, we can also be creating a list. Take the example below:

          $roundness: 20px 0 20px 0;

The "roundness" variable have more than one value, and thus it is in essence a list.

This can be used in different ways. See the example below:

          $roundness: 20px 0 20px 0;
          border-radius: nth($roundness, 1);

The code above tells SASS: - For the border-radius, apply only the first value of the list available in the "roundness" variable, which is 20px.

Also, pay attention that we are using 1 instead of 0. This is because SASS is not zero-based index.

Using SASS to have more control over media queries

When we create media queries, sometimes it can be hard to find exactly what we are looking for if the project gets big. Using SASS we can make this process simple, since we can apply the media query directly on each item. While this can make us spend time with more code, by applying it to every element we want to modify, we also have much better control.

Example:

Create the mixing for the media query:

      @mixin break ($lenght) {
        @media only screen and (max-width: $size) {
          @content;
        }
      }

Then we apply the media query to the code element we want:

      li {
        float: left;
        flex: 1;
        margin: 0.3rem;
        @include break(600px) {
          margin: 0.5rem;
        }
        @include break (1000px) {
          margin: 0.8rem 1.2rem;
        }
      }

Conditional statements in SASS

Like in many programming languages, SASS also let us use conditional statements for the compiler to interpret what you are looking for.

For example, we can use a mixin that takes any number of values. This is called and argument list. To use it we only need to add "..." to the end of a variable.

A good way to use it would be for media queries. Let's take that we need to use "min" and "max" width for our rules, then we could do like below:

      @mixin break (&args...) {
        @if length($args) == 1 {
          @media (min-width: nth($args, 1)) {
            @content;
          }
        }
        @if length($args) == 2 {
          @media (min-width: nth($args, 1)
              and (max-width: nth($args, 2)) {
            @content;
          }
        }
      }

In the example below, "length" is a SASS property (Same as array.lenght), but our variable args tells SASS that we will have more than one argument, then we apply it using the if statement by telling SASS that, if we have more than one arg we will use the second arg also.

See how we could apply the example above to an element:

      pre {
        font-family: $font-code;
        font-size: 1.5rem;
        overflow: auto;
        @include break (400px, 800px) {
          color: pink;
        }
      }

Also, like in other programming languages, it's possible to use the @else statement to avoid more code if it's needed.

Example:

      @mixin break (&args...) {
        @if length($args) == 1 {
          @media (min-width: nth($args, 1)) {
            @content;
          }
        } @else {
          @media (min-width: nth($args, 1)
              and (max-width: nth($args, 2)) {
            @content;
          }
        }
      }
