Working with Math operations

SASS allow us to work with math operations, so we can define styles that have numeric parameters like width, height, margin, etc., and then we can apply them using math.

Check the example below:

          @mixin imageGrid ($qty, $margin) {
            width: ((100% - (($qty - 1) * $margin)) / $qty);

          .grid {
            @include clearfix;
            .item {
              float: left;
              @include imageGrid(3, 1%);
            }
